# Security Policy

## Supported Versions

We actively maintain and support the current major release of the Golito SDK.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of Golito and our users very seriously. If you find a security vulnerability, please do **not** report it via public issues or pull requests. 

Instead, please report it privately to our security team:

* By email: **contact@golito.fun**

Please include as much detail as possible in your report, including:
- A description of the vulnerability.
- A proof of concept (PoC) or steps to reproduce.
- Any potential impact on users or funds.

We will review your submission and respond within **48 hours** to coordinate a fix and release.
